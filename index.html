How to Add Your Data for Hosting on the Website

  1. Visit the repository at https://github.com/kerzendorf-group/group-data and fork it.
  2. Enter the directory named members. Create a new directory with your name in it and inside that directory, create two more directories named media and documents.
  3. Copy the JSON file from an existing person’s directory to your own directory and input your personal details into this file.
  4. Upload your profile picture (square-shaped), which will be displayed on the website, into the media directory.
  5. Upload any other necessary media into the media folder and any documents into the documents directory.
  6. Ensure that after uploading the files, you update your own JSON file present in your directory as well, with the path pointing to the file location. The path to files present inside the media or document directory should start with members/data followed by your directory name and the rest of the path.
  7. In the groupwebsite_data directory, open people_list.json and add a file for your JSON file using the same format as the previously mentioned ones.
  8. Finally, create a Pull Request.

Please note that the “role” key inside personal JSON should be spelled same “Graduates”, “Undergraduates”, “Post Graduates”, “Post Doctorates” only, in order to group people together. You can add "myrole": "Your_role" inside your JSON file for your current role.

Sample template below

[ { "name": "Your name", "project_title": "Project you are currently working in", "org": "Organisation you are in", "category": "write among “Graduates”, “Undergraduates”, “Post Graduates”, “Post Doctorates” only,", "myrole": "Your_role currently", "tags": [], "university": "Michigan State University", "introduction": "Your introduction", "start_year": 2021, "end_year": "None", "image_path": "members/data/ now enter your path", "website": "https://google.com", "twitter": "https://twitter.com", "linkedin": "https://linkedin.com", "email": "fakeemail@gmail.com" } ]
